# Bharat-Sahayak-AI
AI-powered government scheme eligibility and recommendation platform for Indian citizens.

# 🇮🇳 Bharat Sahayak AI

## 🚀 Project Overview

* AI-powered Government Scheme Recommendation System
* Multilingual Support (25+ Indian Languages)
* Personalized Scheme Eligibility Checker
* AI-based Recommendation Engine
* FastAPI + Next.js Full Stack Application

---

# 📌 Workflow

## Step 1: User Access

* Open Bharat Sahayak AI
* Select preferred language
* Register / Login
* Access personalized dashboard

---

## Step 2: User Information Collection

* Personal Details

  * Name
  * Age
  * Gender
* Location

  * State
  * District
* Financial Details

  * Annual Income
  * Category
* Occupation Selection

  * Student
  * Farmer
  * Worker
  * Senior Citizen
  * Widow
  * Self Employed
  * Other

---

## Step 3: Dynamic Form Generation

* Student Details
* Farmer Details
* Worker Details
* Disability Details
* Pension Details
* Additional Category Information

---

## Step 4: Backend Processing

* Validate User Input
* Generate User Profile
* Store Request
* Process Eligibility Criteria

---

## Step 5: Database Operations

* Connect to PostgreSQL
* Retrieve Government Schemes
* Read Scheme Information
* Load Eligibility Rules

---

## Step 6: Eligibility Engine

* Compare User Profile
* Apply Rule-Based Filtering
* Match Eligible Schemes
* Calculate Eligibility Score

---

## Step 7: AI Recommendation Engine

* Analyze Eligible Schemes
* Generate Personalized Recommendations
* Explain Eligibility
* Suggest Best Matching Schemes

---

## Step 8: Multilingual AI Response

* Translate Response
* Generate User-Friendly Explanation
* Display Benefits
* Display Required Documents
* Display Official Application Link

---

## Step 9: Frontend Display

* Show Eligible Schemes
* Display Benefits
* Application URLs
* AI Explanation
* User-Friendly Interface
